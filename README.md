# ECG-triaje-dashboard
Dashboard interactivo en Streamlit para el triaje clínico automatizado de 100 pacientes basado en métricas de ECG.
#Sistema de Triaje Clínico Automatizado (ECG)

Este proyecto es un dashboard interactivo desarrollado en **Python** con **Streamlit** para la gestión, visualización y clasificación de riesgo clínico de un dataset de 100 pacientes basado en sus métricas de ECG.

##Características
- **Triaje Automatizado:** Clasificación en tiempo real (Crítico 🔴, Moderado 🟡, Estable 🟢).
- **Cumplimiento Normativo:** Estructura conceptual alineada a la **NOM-004-SSA3-2012** para el manejo de datos de expedientes clínicos en México.
- **Visualización Interactiva:** Gráficos estadísticos con Plotly para análisis demográfico y fisiológico.

##Tecnologías utilizadas
- Python 3.x
- Streamlit
- Pandas & Plotly
